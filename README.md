## Tổng quan

LocketGold là một dự án chặn và sửa đổi các phản hồi API từ ứng dụng Locket. Mục đích chính của dự án là để giả lập các tính năng của gói thành viên "Locket Gold", cho phép người dùng trải nghiệm các chức năng dành riêng cho những người đăng ký trả phí.

## Tính năng chính

- **Giả lập tính năng "Locket Gold":** Chứa logic để thay đổi hồ sơ người dùng và thông tin tài khoản, can thiệp vào trạng thái `is_gold_member` để kích hoạt các tính năng "Gold".

## Cách sử dụng

### Qua username (Yêu cầu NodeJs):

1. **Tải file javascript**: Tải file [locketGold.js](https://github.com/tuanvuchu/LocketGold/blob/main/locketGold.js) về máy tính
2. **Nhập Tên người dùng**: Nhập tên người dùng Locket mà bạn muốn mở khóa tính năng Gold rồi chạy mã javascript
3. **Thành công**: Nhận phản hồi API khi tính năng Gold đã được mở khóa.

### Qua ứng dụng quản lý proxy (Shadowrocket):

1. **Tải ứng dụng Shadowrocket**: Tải ứng dụng [Shadowrocket (99.000 VNĐ)](https://apps.apple.com/vn/app/shadowrocket/id932747118?l=vi) hoặc có thể liên hệ với tôiii để tải ứng dụng này miễn phí
2. **Thêm cấu hình**: Mở Shadowrocket và thêm cấu hình này:

```bash
https://cdn.jsdelivr.net/gh/tuanvuchu/LocketGold@main/LocketGold.conf
```

3. **Chạy Shadowrocket**: Khởi chạy ứng dụng Shadowrocket
4. **Thành công**: Mở ứng dụng Locket và tận hưởng.

   > Có thể tắt Shadowrocket khi đã mở khóa Gold.
